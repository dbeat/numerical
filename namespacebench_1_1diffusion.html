bool bench::diffusion::fdm_diffusion_a ( )

Computes the 1D diffusion problem defined in reference_diffusion_a() using the finite difference method. Here we solve the problem using the \(\Theta\)-scheme in time and a centered difference scheme in space, i.e.

\[ u_i^{n+1} - \Theta\left(\alpha\left( \frac{u_{i-1}^{n+1}-2u_{i}^{n+1}+u_{i+1}^{n+1}}{\Delta x^2} \right)\right) = \left(1-\Theta\right)\left(\alpha\left( \frac{u_{i-1}^{n}-2u_{i}^{n}+u_{i+1}^{n}}{\Delta x^2} \right)\right) + \Theta\Delta t f_i^{n+1}+\left(1-\Theta\right)f_i^n + u_i^n \]

For \(\Theta=0\) we have the forward Euler scheme in time, for \(\Theta=1\) we have the backward Euler scheme in time and for \(\Theta=1/2\) we have the Crank-Nicolson scheme in time.

In matrix form we have:

\[ \mathbf{A}\mathbf{u} = \mathbf{b} \]

Using the Fourrier coefficient \(F=\alpha\Delta t/\Delta x^2\), the matrix entries are:

\[ A_{i,i-1}=-F\Theta,~A{i, i}=1+2F\Theta,~A_{i, i+1}=-F\Theta \]

.

The vector \(\mathbf{u}\) represents the unknown at the discrete time \(t=t^{n+1}\).

The RHS can be written as:

\[ b_i = u_i^n + F\left(1-\Theta\right)u_{i+1}^n-2u_i^n+u_{i-1}^n + \Delta t \Theta f_i^{n+1} + \Delta t \left(1-\Theta\right)f_i^n \]

For more information see pp. 181-183 of:

R. J. Leveque, Finite Difference Methods for Ordinary and Partial Differential Equations: Steady-State and Time-Dependent Problems, SIAM, 2007.

Returns
The solution.

T bench::diffusion::reference_diffusion_a ( T  x,
T  t,
T  L 
)

Reference solution for the 1D diffusion problem a, defined as:

\[ \frac{\partial u}{\partial t} = -\alpha \frac{\partial^2 u}{\partial x^2} + f(x,t)\quad x \in (0, L),~t \in (0, T] \]

with Dirichlet boundary condition \(u(0, t) = 0\) and \( u(L, t) = 0\) for \(t>0\).

The diffusion coefficient, \( \alpha(L, t) = 1\) is constant and uniform.

We define the source term and initial condition with a reference (manufactured) solution that respects the boundary conditions at the extremities of the interval:

\[ u(x, t) = 5tx*\left(L-x\right) \]

We get the source term substituting the manufactured solution in the diffusion equation, i.e.:

\[ f(x, t) = 5x\left(L-x\right)+10\alpha t \]

The initial condition is simply set to the manufactured solution at \(t=0\):

\[ u(x, 0) = u_0 = 0 \]

LThe length of the interval.

The reference solution.
