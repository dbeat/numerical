A code used to showcase numerical methods in C++ using Eigen3.

Installation
############


1. cd numerical
2. mkdir build && cd build
3. cmake ..
4. make -j4
5. ctest (test)

Documentation available at: https://frnsroy.bitbucket.io/numerical/
